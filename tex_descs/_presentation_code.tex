\section{Présentation du code}
Le code est regroupé dans les fichiers suivants:
\begin{itemize}
\item[projet.py] Regroupe des fonctions utiles pour le chargement des
  données ou pour tester les résultats
\item[data\_texxer.py et results\_to\_tex.py] Regroupe des fonctions
  utile pour (respectivement) décrire les données et décrire les
  résultats en \LaTeX
\item[modele.py] fournit une interface pour les différents
  classifieurs
